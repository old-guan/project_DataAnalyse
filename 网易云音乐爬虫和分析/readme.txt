
项目描述：
1、本项目使用python爬取网易云音乐歌单热榜的数据，以及歌单里的歌曲信息和歌曲评论，并将数据存储到MySQL数据库中；
2、使用sql语句对数据进行分析，并使用pyecharts模块进行可视化；
3、使用jieba模块进行中文分词和词频统计；
4、使用wordcloud模块绘制词云图。

*****************************************************************************************************
1、先执行脚本“createTable.sql”，创建MySQL数据库和数据表，然后执行脚本“createView_*.sql”创建视图；
2、执行python脚本“crawl_music.py”爬取数据；
3、执行python脚本“DataAnalyse.py”进行数据分析、中文分析、词频统计和绘图。
