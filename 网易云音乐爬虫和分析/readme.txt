
项目描述：本项目将爬取网易云音乐歌单热榜的数据，以及歌单里的歌曲信息和歌曲评论，并将数据存储到MySQL数据库中，之后对数据进行分析和可视化，以及词频统计和绘制词云图。

*****************************************************************************************************
1、先执行脚本“createTable.sql”，创建MySQL数据库，然后创建表和视图；
2、执行python脚本“crawl_music.py”爬取数据；
3、执行python脚本“DataAnalyse.py”进进行数据分析和绘图（个别变量需要按实际情况初始化）。
